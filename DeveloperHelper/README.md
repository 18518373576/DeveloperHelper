# androidDevelopHelper
###### 以项目形式记录安卓开发的一些知识点

### 项目环境：

```
ext {
        compileSdkVersion = 28
        buildToolsVersion = "29.0.2"
        minSdkVersion = 21
        targetSdkVersion = 28
    }
```
### jdk:


```
compileOptions 
    {
        targetCompatibility = 1.8
        sourceCompatibility = 1.8
    }
```
